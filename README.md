# grayscale
An app that makes your Mac grayscale/color via accessibility with one click

This package does not work on macOS versions earlier than Mojave.

You can install this package via homebrew. First, tap my homebrew repo (I'm not popular enough for the main repo):

  `brew tap anonymousaga/macapps`
  
Next, install my cask:

  `brew install --cask --force --no-quarantine grayscale`
  

I am always happy to accept help with adding a menu bar icon (which can be clicked) and designing a better icon (this is nearly invisible in grayscale mode).
